# Java Load Testing Tool

A custom **Java-based load testing tool** built from scratch to simulate concurrent users, send multiple HTTP requests, measure performance metrics, and generate CSV reports.

This project is designed for **learning, portfolio, and interview demonstration** purposes.

- Generates **CSV report** (`load-test-result.csv`)
- Config-driven execution (easy to change test parameters)
